# Tracked Tokenised Recycling

Drop it. Earn it. See its impact.

A circular economy platform helping recycling companies track material intake, reward recyclers with tokens or Bitcoin sats, and build verified environmental impact records.

## Core Features

- Recycler rewards
- Operator drop-off logging
- Company dashboards
- AI tools
- Bitcoin rewards
- Verified audit trail

## Stack

Frontend: Next.js  
Backend: FastAPI  
Database: PostgreSQL  
AI Integration 
Bitcoin: Kotani Pay / LNbits

## Status

Early MVP build in progress.
